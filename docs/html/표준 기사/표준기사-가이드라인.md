# 표준 기사 가이드라인 (경기 기사) — Swimming Photography

> 기사작성규약 v2.0(agreement.html)을 경기 기사 1편 작성에 바로 쓸 수 있게 압축한 실무 가이드.
> 괄호의 「제○조」는 규약 조항 번호. 충돌 시 규약이 우선한다.

---

## 0. 기사 데이터 라인 (본문 앞 메타)

```
[등록구분] 성별 영법 거리 코스 | 기록 | 선수명 | 0000년 0월 0일 요일 | 광역, 수영장명
예) 비등록 여자 접영 50m LCM | 32초 01 | 김희경 | 2026년 5월 31일 일요일 | 경기, 고양체육관 수영장
```
- 이 라인은 기사 상단 메타·사이드바 기록 카드용. 본문 문장이 아니므로 규약의 문장 규칙을 적용받지 않는다.

## 1. 제목·부제 (제1조)

- 제목 **25자 이내**, 부제 **30자 이내**. 사실 중심, 명사 종지 또는 현재형 종지(과거형은 본문에서).
- 보조 정보는 약물 기호 「…」 뒤 (마침표 3개 금지).
- 금지: 물음표·느낌표·이모지 / 감정형 부제 / 「충격」·「대박」류 가치어 / 근거 없는 「~할 듯」.

## 2. 본문 구조 — 역삼각형 4~5단락 (제2·3조)

| 단락 | 내용 | 핵심 규칙 |
|---|---|---|
| ① 리드 | 누가·무엇·언제 (첫 문장 60자 이내) → 어디·대회 → 왜·어떻게 | 리드만 읽어도 전모 파악. 감상·배경으로 시작 금지 |
| ② 인용 | 경기 소감 등 직접인용 | 한 단락에 직접인용 1개 (길면 단락 분리) |
| ③ 배경 | 선수 이력·스타일·맥락 | **사실만.** 평가·해석은 평가 주체 명시 |
| ④ 부연 | 훈련·소속·경기 세부 | 한 단락 = 소주제 1개, 2~4문장 |
| ⑤ 마무리 | 대회 맥락·확인된 향후 일정 | 미확인 일정·전망 금지 |

- 한 문장 60자 이내·사실 1개. 능동형 우선(「수립했다」).
- 끝 단락부터 잘라내도 기사가 성립해야 한다.

## 3. 문체 금지 요약 (제3.4·7·8조)

- **1인칭 금지**: 「나·저·우리」, 기자 감사·소감·박수 → 삭제하거나 취재원 인용으로 전환.
- **강조 부사 금지**: 매우·정말·너무·엄청·굉장히·참으로·실로·그야말로.
- **가치 형용사·과장 금지**: 훌륭한·감격적인·역대급·공룡급·만리장성처럼 등 비유 과장 포함.
- **광고성 금지**: 시설·장비·브랜드 일방 칭찬. → 관찰 가능한 사실로 서술(「LED 전광판이 설치됐다」).
- **추측 금지**: 「~로 보인다」·「~였으리라 예상된다」는 출처 명시 없이는 불가.
- **어제·오늘·내일 금지**(본문). 올해·작년·내년 대신 연도를 명기. 인용 안은 예외.
- 「하였다→했다」 「되었다→됐다」 축약. 접속사 뒤 쉼표 금지.

## 4. 표기 (제12·13·20조)

- **인명 첫 등장** 「김희경(나이·소속) 선수」 → **재등장** 「김 선수」. 호칭은 띄어쓰기.
- **기록** 「32초 01」 「1분 54초 22」 — 한글 단위·띄어쓰기·1/100초 두 자리(53초 00).
- **종목** 「접영 50m」 — 영문 병기 금지(코스 LCM 등은 데이터 라인에만).
- **날짜** 「5월 31일」 (보도 연도·월 생략 가능). 「지난·오는」 생략.
- **대회명** 첫 등장 정식 명칭 → 이후 약칭.
- **신기록** 「한국신기록(KR)」 + 단축 폭 명시. **순위** 「1위」·「금메달」 허용.
- 열거 3개 이상은 「△」. 복합어는 가운뎃점(·).

## 5. 인용 (제4조)

- 인용문은 신성불가침 — 취지 훼손 없는 축약·정리만 허용(구어 존댓말→평서 정리 가능, 의미 변경 금지).
- 인용 동사는 「말했다」 기본. 「~라고 말했다」(종결) / 「~라며 …했다」(연결) 구별.
- 직접 인용문 끝에 마침표를 찍지 않고, 따옴표 뒤에 찍는다: 「"선방했다고 생각한다"고 말했다.」

## 6. 사진·캡션 (제5·9·15조)

- **본인이 직접 촬영한 사진만**, 기사당 **1~5장**. 촬영일·장소·카메라 정보 필수 입력.
- 캡션 2문장: ① 현재 진행형 동작 묘사 ② 과거형 맥락·결과. 5요소(인물·동작·일시·장소·대회) 포함.
- 형식: 「[설명]. 사진=촬영자명 / 2026. 5. 31. 고양체육관 수영장」 (편집부 촬영 시 「사진=편집부」).
- 다인 사진은 「왼쪽부터」. 「포즈를 취하고 있다」류 무의미 동사 금지.

## 7. 바이라인 (제6조)

- 객원기자: 「수영사진 ○○○ 기자 / ooo@medalbank.com」
- 편집부: 「수영사진 편집부 / press@medalbank.com」 (제보 실명 시 「/ 제보 ○○○」)

## 8. 분량·분류 (제10·11조)

- 본문(헤드라인·캡션·바이라인 제외) **1,000~1,800자**. 600자 이하는 단신 처리.
- 분야 1개(경기/인물/현장/칼럼/사진) + 광역 1개(예: 경기) + 세부 태그(예: 고양시).

## 9. 캐주얼 문체 → 표준 전환 원칙

기존에 써 온 캐주얼 후기체를 표준 기사로 바꿀 때의 변환 규칙:

| 캐주얼 (예전) | 표준 (전환) |
|---|---|
| "한 명의 수영인으로서 감사하다" (기자 소감) | 삭제, 또는 참가 선수 인용으로 |
| "훌륭한 운영에 박수갈채를 보낸다" | 「경기는 지연 없이 진행됐다」 등 관찰 사실 |
| "만리장성처럼 펼쳐진 LED 전광판" | 「출발대 뒤로 대형 LED 전광판이 설치됐다」 |
| "배럴·아레나급 공룡 대회를 방불케" | 삭제 (브랜드 비교·과장) |
| "물이 시원하고 좋은 경기장으로 손꼽히며" | 「수온을 ○○도로 유지했다」 (수치) 또는 참가자 인용 |
| "분명 ~였으리라 예상할 수 있다" | 삭제, 또는 주최 측 확인 후 인용 |
| "올해는 / 내년엔" | 「2026년 대회는 / 2027년 대회는」 |
| "많은 선수들이 만족스러운 표정" | 「참가 선수 ○○○ 씨는 "…"라고 말했다」 |

> 요령: **느낀 것은 버리거나 사람 입으로 옮기고, 본 것은 수치·사실로 적는다.**
