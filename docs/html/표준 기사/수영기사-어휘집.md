## 4. 레이스 전개 서술 (현장 관찰 재료가 있을 때만)

- 초반부터 거세게 질주하며 / 초반부터 스퍼트를 올리는 전략을 펼쳤다
- 첫 50m 구간부터 힘을 내며 곧바로 1위로 치고 나갔다
- 50m를 돌아선 뒤 1위로 올라섰고, 300m 지점까지 독주했다
- 물살을 갈랐다
- 한 번도 선두를 내주지 않고 / 시작부터 끝까지 줄곧 선두를 유지하며
- 이후 속도가 떨어졌지만 초반 격차의 우위를 바탕으로
- 4위까지 처졌으나 300m 구간을 돌 때 3위로 올라섰고 이후 계속 자리를 지켜
- 마지막 50m에서 역전을 허용했지만 막판 추격은 뿌리쳤다
- 경쟁자들을 제쳤다
- 경기 시작과 동시에 속력을 높인 ○○○을 따라 2위를 유지하던
- 기록을 확인하고 기뻐하고 있다 (사진 캡션)

### 4-1. 구간(스플릿) 기반 표현 — **`splits` 데이터가 있을 때만** (SS)

> 마스터즈 50m 단일 종목은 스플릿이 나오지 않는다. **100m 이상 종목 기사에서만** 쓴다.
> 스플릿이 없으면 이 표현들을 쓰지 않는다 — 구간을 추정해 지어내지 않는다.

- 초반을 강하게 가져갔다
- 초반부터 빠르게 나가 100m를 59초 70에 통과했다

## 5. 인명 지칭 관례 — ⚠️ 수집 10편 전수 확인

**재등장은 풀네임 반복이다. 「김 선수」·「강 선수」식 성+선수 축약은 10편 어디에도 없다.**

| 관례 | 실제 용례 |
|---|---|
| 첫 등장 = 이름(나이·소속) | 「김우민(22·강원도청)이 박태환 이후 13년 만에…」(한겨레) 「김우민(24·강원도청)이…」(경향) |
| 재등장 = **풀네임** | 「김우민은 12일…」 「이날 김우민은 오전 예선에서…」 「결승에서 김우민은…」 — 전 매체 동일 |
| 「○ 선수」 호칭 | 본문 서술에는 안 쓴다. 방송 인터뷰 자막(「김우민/수영 국가대표」)과 발언 인용 안에서만 |
| 반복 완화 = 수식어 앞세우기 | 「예선을 마친 김우민은」 「경기 뒤 감격의 눈물을 흘린 김우민은」 「1번 레인에서 출발한 김우민은」 — 대명사·축약이 아니라 **문두 수식어로 변주**한다 |
| 경쟁 선수 = 순위+이름(정보) | 「2위 일라이자 위닝턴(23·호주·3분42초86)을 0.15초 차로 따돌렸다」 「3위는 루카스 마르텐스(22·독일·3분42초96)」 |
| 별칭 | 「'마린보이' 박태환」 — 통용되는 별칭만, 창작 금지 |

> **본 매체 적용**: 첫 등장만 「강의신(43·스윔온) 선수」로 쓰고, **그 뒤로는 전부 풀네임** 「강의신은」·「강의신의」. 「강 선수」를 쓰지 않는다.
> (영문은 기존 규칙 유지 — 첫 등장 `KANG, Uisin`, 후속 성만 `KANG`.)

## 7. 쓰지 않을 표현 (일간지엔 있으나 본 매체 규약 위반)

실제 기사에도 등장하지만 **주체 없는 평가·전망**이므로 본 매체에서는 **버린다.**

| 원문 예 | 판정 |
|---|---|
| 「금메달 획득 기대감도 커졌다」 | 주체 없는 전망 → 삭제 |
| 「가파른 성장세를 보이고 있다」 | 주체 없는 평가 → 삭제 |
| 「~란 분석입니다」 | 분석 주체 없음 → 삭제. 주체 있으면 인용으로 |
| 「한국 수영이 경쟁력을 유지하고 있음을 보여주는 결과다」 | 기자 해석 → 삭제 |
| 「새 기록을 쓸지도 주목됩니다」 | 전망 → 삭제 |
| 「넘어설 가능성도 있습니다」 | 추측 → 삭제 |
| 「무려 15초 이상」 | 「무려」는 강조 부사 → 「15초 이상」만 |
| 「아쉽게도」·「힘겹게」 | 감상 수식 → 삭제 |

> 즉 **동작·전개·결과 어휘는 그대로 차용하고, 평가·전망 어휘는 전부 버린다.**

## 11. 제목·부제 — 긍정 사실만 (2026-07-18 확정)

기사에 실리는 것 자체가 선수에게 명예로운 일이다. 성적을 앞세운 제목은 **비꼬거나 비웃는 것처럼 읽힐 수 있다.**

| 구분 | 제목·부제에 쓸 수 있는 것 | 쓰지 않는 것 |
|---|---|---|
| 순위 | 1위·2위(은메달)·3위(동메달), 2관왕 등 복수 입상 | **4위 이하 순위 전부** |
| 기록 | 개인 최고기록 경신, 기록 자체(「45초 81」) | 기록 미달·기권·실격 |
| 출전 | 출전·역영 사실(「두 종목 역영」) | 다른 선수와의 격차, **「완주」** |

- 입상이 하나도 없으면 **순위 대신 기록이나 「역영」**을 제목으로 삼는다.
- ⚠️ **제목·부제에 「완주」 금지.** 경영에서 완주는 당연한 일이라 제목감이 아니다. → 「역영」으로 대체.
  - ✕ 「김병한, 평영 50m·100m 완주」 → ○ 「김병한, 평영 50m·100m 역영」
  - ✕ 「이원갑, 접영 6위·자유형 11위」 → ○ 「이원갑, 접영 50m 31초 95」
  - ✕ 「29초 52… 자유형 50m는 4위」 → ○ 「29초 52… 자유형 50m 28초 03」
- 본문에서는 순위를 사실대로 쓰되 **열위를 강조하는 프레이밍을 하지 않는다.**
  - ✕ 「전체 1위 ○○○의 31초 83에 6.08초 뒤졌다」 → ○ 「전체 1위는 ○○○의 31초 83이었다」
  - 단 **1~3위 선수**에게는 격차 서술(「0.15초 차로 따돌렸다」)을 그대로 쓸 수 있다.
- 「시상대에 오르지 못했다」는 폐기(10항). 부정 프레이밍이다.

## 13. 제목 전용 어휘 (담당자 지정, 2026-07-18)

담당자가 제목에 쓰라고 지정한 어휘. **제목·부제 전용**이며, 본문에서는 기존 규칙(0항 감상 금지)이 그대로 적용된다.

### A. 사실 어휘 — 조건 없이 사용

| 어휘 | 쓰는 자리 |
|---|---|
| **기록** | 「접영 50m 31초 95 기록」 「대회 최고 기록」 |
| **금메달** | 1위일 때 |
| **메달리스트 / 금메달리스트** | 1~3위 / 1위. 이력을 함께 쓸 때 유용 |
| **입상** | 1~3위를 뭉뚱그릴 때. 종목이 여럿일 때 편하다 |
| **연속 입상** | **직전 대회 입상 기록이 데이터에 있을 때만** |
| **획득** | 「금메달 획득」 「메달 2개 획득」 |
| **결승 / 결선** | 라운드 표기. 본 대회 기록지 표기(결선)를 따르되, 제목에서는 둘 다 쓸 수 있다 |

### B. 수사 어휘 — **사실 근거가 있을 때만**

아래 넷은 **평가·비유**여서 본문 금지어와 성격이 같다. 담당자 지정으로 **제목에 한해** 허용하되,
근거 없이 붙이면 0항 4번(감상 금지)과 정면으로 충돌한다. **반드시 아래 조건을 만족할 때만 쓴다.**

| 어휘 | 쓸 수 있는 조건 | 예 |
|---|---|---|
| **금빛물살** | 금메달을 딴 경기에 한해 | 「금빛물살 가른 정대교, 개인혼영·접영 2관왕」 |
| **쾌거** | 기록 경신·첫 입상·복수 관왕 등 **객관적으로 특별한 성과**가 있을 때 | 「정대교 2관왕 쾌거」 |
| **희망** | 미래 전망이 아니라 **확인된 다음 일정·성장 사실**과 함께 | 남용 주의 |
| **황금시대** | 팀·대회 단위로 **누적 성적 근거**가 있을 때 | 「GPG, 메달 79개의 황금시대」 |

> ⚠️ **B군은 한 대회에서 한두 건까지만.** 모든 제목에 붙으면 매체 신뢰가 떨어지고, 입상하지 못한 선수 기사와 나란히 놓였을 때 상대적으로 깎아내리는 효과가 난다(5항).
> ⚠️ B군을 **본문에 쓰지 않는다.** 본문은 사실만.
